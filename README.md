# Color Apps In C# .NET WinForms
Color Apps developed using C#.NET

There are two apps in Color Apps:

## Simple Color Palette Generator
Simple Color Picker software is a tool to generate color palette containing six (6) colors. 
It can also provide the conversions of RGB color to HEX, CMYK, HSV and HSL. 
The user can save all the selected colors in a file.

## Simple Color Picker
Simple Color Picker software is a tool to capture the color from the color spectrum. 
It can also provide the conversions of RGB color to HEX, CMYK, HSV and HSL.
